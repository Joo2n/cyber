# 심플 사이버연수원 화면 설계 가이드

## 1. 디자인 철학

### 1.1 핵심 원칙
- **단순함(Simplicity)**: 불필요한 요소 제거, 핵심 기능에 집중
- **직관성(Intuitiveness)**: 사용자가 생각하지 않아도 사용할 수 있는 인터페이스
- **일관성(Consistency)**: 모든 화면에서 동일한 패턴과 규칙 적용
- **접근성(Accessibility)**: 모든 사용자가 쉽게 접근할 수 있는 디자인

### 1.2 참고 사이트 분석
**bestez.wowpass.com 특징**:
- 깔끔한 로그인 중심 레이아웃
- 명확한 네비게이션 구조
- 학습 지원 정보 우측 배치
- 차분한 색상 팔레트
